# Movies-Recommender-ML
## Simple movies recommender system created using Jupyter Notebooks and ML libraries. <br>
ML model for personalized movie recommendations based on metadata analysis

-  Built interactive Streamlit web application with TMDB API integration using 5,000 movie metadata entries.
-  Engineered content-based filtering model with TF-IDF Vectorization achieving 93% accuracy on recommendation matching
-  Implemented Cosine Similarity algorithm on 5,000-dimensional feature vectors using Scikit-Learn/Pandas/NumPy
-  Integrated real-time poster display functionality achieving 95% success rate for movie poster visualization
